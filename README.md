# ЭлектроСмета

PWA-приложение для расчёта смет на электромонтажные работы.

## Функции

- Каталог электромонтажных услуг с ценами
- Формирование сметы с расчётом итоговой суммы
- Экспорт сметы в PDF
- Работа офлайн (PWA)
- Тёмная/светлая тема

## Технологии

- Next.js 16
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zustand
- pdf-lib

## Разработка

```bash
bun install
bun run dev
```

## Сборка

```bash
bun run build
```
